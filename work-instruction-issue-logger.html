Work Instruction: Issue Logger

FWA Operations — CTDI Coatesville

1. Purpose

This work instruction describes how to use the Issue Logger to record device/product issues during receipt inspection. The tool supports both individual entry and high-speed batch scanning modes.

2. Before You Begin

  • Navigate to Product/Device Issues → Issue Logger
  • Ensure your barcode scanner is connected
  • Confirm the connection status indicator shows green (Connected)
  • Verify the correct Cart is available in the dropdown

3. Individual Entry Mode

Logging a Single Issue

  1. Fill in the IMEI field (15 characters, starts with '3')
  2. Fill in the Serial Number field (11 characters, starts with 'A')
  3. Select or scan the Cart
  4. Select the Device Model (Titan 3 or Titan 4)
  5. Enter Reported By (your name)
  6. Enter or select an Issue Note describing the problem
  7. Click "Log Issue"
Tip: When you select a cart, the Issue Note auto-fills with the cart's standard note. You can edit it if needed.

4. Batch Scan Mode

Batch mode is designed for high-speed scanning of multiple units with the same Cart, Device Model, Reported By, and Note values.

Starting Batch Mode

  1. Fill in Cart, Device Model, Reported By, and Issue Note on the main form
  2. Click "Start Batch Scan"
  3. These values are locked for the batch session

Scan Mode Toggle

At the top of batch mode, choose your scan method:

ModeHow It WorksBest For
Scan Values IndividuallyScan IMEI (15 chars) → auto-advances to Serial (11 chars) → auto-submitsUnits with separate IMEI and Serial barcodes
Scan QR CodeScan unit QR code → auto-parses IMEI, Serial, and Model → auto-submits on EnterUnits with QR codes containing all info

Scanning Units (Individual Mode)

  1. Scan the IMEI barcode — cursor auto-advances to Serial field after 15 chars
  2. Scan the Serial Number barcode — auto-submits after 11 chars
  3. A success beep plays and the fields clear for the next unit
  4. The "This Batch" counter increments
  5. Repeat for each unit

Scanning Units (QR Code Mode)

  1. Select "Scan QR Code" at the top
  2. Scan the unit's QR code — press Enter or let the scanner send Enter
  3. The system automatically extracts IMEI, Serial, and Device Model
  4. A success beep plays and the field clears for the next unit
  5. Device Model updates automatically based on the QR data
Note: You can switch the Device Model dropdown between scans in both modes.

Deleting a Scanned Record

  1. In the batch log table, click "Delete" next to the record
  2. Confirm the deletion
  3. This removes the issue log entry and removes the unit from the cart
Important: Deleting a record removes it from both the Device Issues sheet and the Device Location (cart) sheet.

Exiting Batch Mode

  1. Click the red "Exit Batch Mode" button
  2. You return to the individual entry form
