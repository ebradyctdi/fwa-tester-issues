Setup Instructions

  1. Create a new Google Sheet with a tab named "Tester Issue Log"
  2. Add headers: Reported By | Tester Type | Tester ID | Severity | Issue Type | Description | Status | Issue Start | Issue Stop
  3. Go to Extensions → Apps Script
  4. Paste the code from google-apps-script.js
  5. Deploy → New Deployment → Web App → Execute as: Me → Who has access: Anyone
  6. Copy the URL and paste it above
